图像标记

![1524396193568](.\images\网页基础知识2图1.png)

src用于指定图像文件的路径及文件名

|  属性  | 属性值 |           描述           |
| :----: | :----: | :----------------------: |
|  src   |  URL   |        图像的路径        |
|  alt   |  文本  | 图像不能显示时的替换文本 |
| title  |  文本  |    鼠标悬停时显示内容    |
| width  |  像素  |       设置图像宽度       |
| height |  像素  |       设置图像高度       |
| border |  数字  |     设置图像边的宽度     |
| vspace |  像素  |  设置图像顶部和底部空白  |
| hspace |  像素  | 设置图像左侧和和右侧空白 |
| align  |        |                          |
